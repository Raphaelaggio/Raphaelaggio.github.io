Metab

Metab is an R package developed to process metabolomics data in a high-throughput manner. Metab processes data previously analysed by the Link textAutomated Mass Spectral Deconvolution and Identification System (AMDIS) and allows users to perform data normalisation and statistical analyses. Metab has been accepted by Bioconductor but will be available only in the next release of Bioconductor database. In the meanwhile, the source of Metab can be downloaded here.

For installing Metab, place the source file you just downloaded in your working directory, copy the code below and paste it at your R console:


### Install required packages
source("http://bioconductor.org/biocLite.R")
biocLite(c("xcms", "svDialogs", "pander”))
### Install Metab
install.packages(“Metab_0.99.7.tar.gz", type = "source")

PAPi

PAPi is an R package developed to predict the activity of metabolic pathways based solely on metabolomics data sets. Based on the number and abundances of metabolites identified per metabolic pathway, PAPi calculates the activity score of metabolic pathways in the Kyoto Encyclopedia of Genes and Genomes (KEGG). PAPi is available at Bioconductor

Alternativelly, you can install PAPi direct from your R console. Just copy the code bellow and paste it at your R console:


source("http://bioconductor.org/biocLite.R")
biocLite("PAPi")

MetaBox

MetaBox is an R package developed to identify compounds analysed by GC-MS by scoring chromatographic peaks according to their likelihood of representing metabolites defined in a mass spectral library. MetaBox has been submitted for publication and will be available for downloading soon.
